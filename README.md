## Motivation

Customized Hexo for me.

## Usaga

### 1. Install [Hexo](https://github.com/hexojs/hexo).
### 2. Update dependency

Change your `package.json` below.

```
"hexo": "git+https://github.com/YoshinoriN/Nuxa.git#master",
```
